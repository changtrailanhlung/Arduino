# Check_temperature_people
Module:
- LCD 16x02
- I2c (for LCD 16x02)
- Sensor MLX90614 (infrared thermometer for non-contact temperature measurement)
- Sensor HC-SR04
- DFPlayer Mini
- Speaker
- Arduino Nano
- Led Red, Yellow, Green (5V)
- Switch
- Jack DC
- Transistor 1k
- PAM8403 3Wx2 Audio Amplifier Circuit
